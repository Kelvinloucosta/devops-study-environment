Step-by-step guide

From zero to isolated DevOps env

01

Install Homebrew

The foundation. Every other tool installs and uninstalls through it cleanly.

# Paste this in Terminal
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
# After install, follow the prompt to add Homebrew to your PATH
# On M-series it'll look like:
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc
02

Install OrbStack — Docker + Kubernetes

Replaces Docker Desktop. Boots in seconds, uses a fraction of the RAM, and ships with a built-in K8s cluster.

brew install --cask orbstack
# Verify both Docker and kubectl are available
docker version
kubectl version --client
Open OrbStack from Applications, enable Kubernetes in its settings, and you have a single-node cluster ready to go.
03

Install UTM — Linux VMs

For when you need a real Linux environment with systemd, package managers, or anything containers can't simulate.

brew install --cask utm
Download ARM64 images only. For Ubuntu Server: ubuntu.com/download/server/arm. UTM also has a built-in gallery with pre-configured VMs.
04

Install mise — per-project tool versions

Keeps kubectl, terraform, helm, and other CLIs versioned per project so your study setup never bleeds into work.

brew install mise

# Add to ~/.zshrc
echo 'eval "$(mise activate zsh)"' >> ~/.zshrc
source ~/.zshrc
# Inside your study project folder:
mise use kubectl@1.29
mise use helm@3.14
mise use terraform@1.7

# This creates a .mise.toml file — versions activate only in this folder
05

Install direnv — env vars per folder

Automatically loads and unloads environment variables when you enter or leave a directory. Keeps contexts isolated.

brew install direnv

# Add to ~/.zshrc
echo 'eval "$(direnv hook zsh)"' >> ~/.zshrc
source ~/.zshrc
# Create a .envrc in your study folder:
cat > ~/studies/k8s/.envrc << 'EOF'
export KUBECONFIG=~/.kube/config-lab
export AWS_PROFILE=lab
export ENVIRONMENT=dev
EOF

direnv allow  # run once to trust the file
From now on: cd ~/studies/k8s loads the env vars. cd .. removes them automatically.
06

Install act — run GitHub Actions locally

Runs your .github/workflows files locally using Docker, without pushing to GitHub.

brew install act
# In any repo with GitHub Actions workflows:
act -l            # list available workflows
act push          # simulate a push event
act -j build      # run a specific job
Requires OrbStack (Docker) to be running.
07

Create your study folder structure

Keep everything study-related in one place so removal is trivial later.

mkdir -p ~/studies/{docker,k8s,terraform,github-actions,linux}

# Each folder gets its own .mise.toml and .envrc
# so tools and env vars never cross-contaminate

Clean teardown

Removal guide

Remove everything cleanly when you want to reclaim disk space or start fresh. Follow this order to avoid orphaned config files.

Heads up: Removing UTM VMs and Docker images can free up 10–30 GB. Always check before deleting anything you still need.
Tool Uninstall command Extra cleanup
act brew uninstall act Remove pulled Docker images with docker images | grep act | awk '{print $3}' | xargs docker rmi
direnv brew uninstall direnv Remove eval "$(direnv hook zsh)" from ~/.zshrc. Delete ~/.config/direnv
mise brew uninstall mise Remove eval "$(mise activate zsh)" from ~/.zshrc. Delete ~/.local/share/mise and ~/.config/mise
OrbStack brew uninstall --cask orbstack Delete ~/OrbStack and ~/Library/Application Support/OrbStack to remove all images and VMs
UTM brew uninstall --cask utm Delete VMs: rm -rf ~/Library/Containers/com.utmapp.UTM and ~/Documents/UTM
Homebrew /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/uninstall.sh)" — run this last, after everything else is gone

Full cleanup in one pass

# Remove tools (in order)
brew uninstall act
brew uninstall direnv
brew uninstall mise
brew uninstall --cask orbstack
brew uninstall --cask utm

# Remove data directories
rm -rf ~/OrbStack
rm -rf ~/Library/Application\ Support/OrbStack
rm -rf ~/Library/Containers/com.utmapp.UTM
rm -rf ~/Documents/UTM
rm -rf ~/.local/share/mise
rm -rf ~/.config/mise
rm -rf ~/.config/direnv

# Clean .zshrc — remove these three lines:
# eval "$(/opt/homebrew/bin/brew shellenv)"
# eval "$(mise activate zsh)"
# eval "$(direnv hook zsh)"

# Finally, remove Homebrew itself
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/uninstall.sh)"
